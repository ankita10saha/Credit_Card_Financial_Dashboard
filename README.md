# Credit_Card_Financial_Dashboard
Credit Card Customer & Transaction Analysis
Power BI Dashboard

Overview:
End-to-end data analytics project analyzing credit card customers and transactions using SQL and Power BI to deliver actionable business insights.

Objective:
Analyze revenue, transactions, and customer behavior.
Identify top card categories, spending patterns, and key regions.
Track activation, delinquency, and time-based trends.

Tools & Skills:
SQL- Data cleaning, joins, aggregations, CTEs
Power BI- Data modeling, interactive dashboards
DAX- KPIs, WoW & YTD metrics
